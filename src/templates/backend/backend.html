Welcome to the applications Backend.

You can either work with existing collections or generate new collections using file upload.

Using existing Collections

Choose Collection
Choose an existing collection in the database and explore the specific data in the application.
Edit Collection
Edit Metadata of an existing Database Collection.
Explore Collection
Explore Metadata of an existing Database Collection.
Remove Collection
Remove existing Collections from servers internal database.

Create new Collections

Upload Files
Before creating a new collection, a trace in pcap format has to be uploaded to the server.
Remove Files
Remove existing pcap-files from the server in case they are not needed anymore.
Store Collection
Select an uploaded pcap file an load it into the database. Afterwards edit the collections metadata.

Real-Time Monitoring

Local Monitoring
Choose a local interface to monitor on. Please be sure that this interface is configured appropriately.
Remote Monitoring
Choose a configured Remote Connection to monitor. The Remote Machine needs a key to authenticate the server. Please refer ssh-keygen and ssh-copy-id.
Configure Connections
Configure, List and Delete available Remote Connections. Key distribution is NOT included.
Status and Management
Explore running monitoring tasks, including canceling or deleting a task. This information is not available after Server Reboot.
Returning back to Instructions will give an overview of existings functionality.
